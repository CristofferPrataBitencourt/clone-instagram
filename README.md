# 📸 Instagram Clone Interface

Um desafio de desenvolvimento front-end focado na replicação da interface do Instagram. O objetivo foi aplicar técnicas avançadas de CSS para atingir a máxima fidelidade visual ao design original.

## 🎯 Objetivos do Projeto
* **Mobile First:** A interface foi pensada primeiro para dispositivos móveis, seguindo a essência da rede social.
* **Componentização:** Estruturação de posts, stories e barras de navegação.
* **Fidelidade Visual:** Estilização de bordas, sombras e tipografia idênticas à plataforma.

## 🛠️ Tecnologias
* HTML5
* CSS3 (Variáveis e Flexbox)
